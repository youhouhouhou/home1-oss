oss-lib-webmvc

对spring-webmvc进行一些配置, 提供支持web应用的小工具.

对spring-webmvc进行的配置主要是在数据序列化反序列化方面, 在我们的微服务环境下, 各个服务之间主要通过RESTful风格的HTTP接口进行通信。 如果每个服务对数据序列化和反序列化的细节不完全一致，会导致灾难。

如果用户在调用第三方API时需要定制的序列化/反序列化行为，请在专门的客户端上单独配置，不应破坏全局的设置。
