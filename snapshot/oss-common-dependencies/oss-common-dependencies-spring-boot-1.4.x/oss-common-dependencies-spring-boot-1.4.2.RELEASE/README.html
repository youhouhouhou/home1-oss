oss-common-dependencies-spring-boot-1.4.2.RELEASE

当你使用spring-boot-1.4.2.RELEASE时, 应该使用oss-common-dependencies-spring-boot-1.4.2.RELEASE的pom管理你的依赖.

本pom继承自oss-common-dependencies-spring-boot-1.4.x, 当oss-common-dependencies-spring-boot-1.4.x定义的依赖版本与spring-boot-1.4.2.RELEASE不兼容时, 可以通过<properties>或<dependencyManagement>覆盖parent的设置.
